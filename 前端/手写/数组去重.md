# 给定一个数组，里面包含重复的数字或字符串，请写一个函数返回去重后的数组。

``` Javascript
// es6 ... 扩展运算符
const unique = (arr) => {
  return [...new Set(arr)]
}

const unique = (arr) => {
  const hash = new Map();
  const res = [];
  for (const val of arr) {
    if (!hash.has(val)) {
      hash.set(val, true)
      res.push(val);
    }
  }
  return res;
}
```
